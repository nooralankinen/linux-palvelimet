- Komentokehote, ylläpito ja paketinhallinta
  
Demonit
- Apache-weppipalvelin
- Syventävä aihe weppipalvelimista
- OpenSSH-etähallinta.
  
Automatisointi
- Automaatio ja ohjelmointiympäristöt
